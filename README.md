## Описание программы

Weather Diary — это графическое приложение для ведения дневника погоды. Позволяет добавлять, просматривать и фильтровать записи о погоде, сохранять данные в JSON файл и загружать их обратно.

**Основные функции:**
- Добавление записей о погоде с датой, температурой, описанием и информацией об осадках
- Просмотр всех записей в табличном виде
- Фильтрация записей по дате и температуре
- Сохранение данных в JSON файл
- Валидация вводимых данных
- Удаление выбранных записей

## Требования для запуска

- Python 3.8 или выше
- Библиотека: `tkinter` (стандартная, устанавливается с Python)

## Как запустить

```bash
# Клонировать репозиторий
git clone https://github.com/radmirhub/Weather-Diary-1.git
cd Weather-Diary

# Установить зависимости
pip install -r requirements.txt

# Запустить приложение
python main.py
```

---

## Как пользоваться

1. **Добавление записи:**
   - Введите дату в формате ГГГГ-ММ-ДД (например, 2026-04-30)
   - Введите температуру в градусах Цельсия (например, 22.5)
   - Введите описание погоды (например, "Солнечно")
   - Выберите наличие осадков ("да" или "нет")
   - Нажмите кнопку "Добавить запись"

2. **Фильтрация записей:**
   - Для фильтрации по дате: введите дату в поле "Фильтр по дате"
   - Для фильтрации по температуре: введите минимальное значение в поле "Фильтр по температуре (>)" 
   - Нажмите "Применить фильтр"
   - Для сброса фильтра нажмите "Сбросить"

3. **Управление данными:**
   - Данные автоматически сохраняются в файл `data.json`
   - Для принудительного сохранения нажмите "Сохранить в JSON"
   - Для загрузки данных нажмите "Загрузить из JSON"
   - Для удаления записи выберите её в таблице и нажмите "Удалить выбранную"
